## 🚀 如何使用

### 啟動服務
```bash
python start_server.py
```

### 測試 API
```bash
curl -X POST http://localhost:8000/tag-cover \
  -F "file=@your_image.jpg" \
  -F "top_k=5" \
  -F "confidence_threshold=0.3"
```

### 添加參考圖片到 RAG
```bash
curl -X POST http://localhost:8000/rag/add \
  -F "file=@reference_image.jpg" \
  -F 'tags=["蘿莉", "校服"]' \
  -F 'metadata={"description": "Loli in school uniform"}'
```
